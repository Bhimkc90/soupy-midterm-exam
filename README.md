# Soupy Midterm Exam

A static e-commerce website developed for the CSCI 355 Internet and Web Technology midterm examination.

## Features

- HTML5 page structure
- CSS styling and layout
- Product catalog
- Quantity selector (+/-)
- Add to Cart functionality
- Shopping cart counter
- Social media icons
- Product image gallery

## Technologies Used

- HTML
- CSS
- JavaScript
